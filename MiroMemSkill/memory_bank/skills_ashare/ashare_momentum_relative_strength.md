## 步骤

1. **多窗口相对收益**：用 ashare_price_history 与 ashare_index_history（as_of 严格取题目日期，**lookback_days=250**）取尽可能长的历史，分别计算个股与指数的 5/20/60/120 日收益率，得到多窗口的**相对（超额）收益**。
2. **趋势一致性判别**：
   - 多窗口超额同号且递增 → 趋势延续概率高，顺势预测；
   - 短期(5日)与中长期(20/60/120日)背离 → 警惕反转，降低置信度；
   - 20 日超额在 ±2% 内 → 视为噪声区间，需依赖估值/基本面辅助判断。
3. **量价确认**：换手率放大且价格上行 → 增量资金；缩量上涨/放量下跌 → 持续性存疑。
4. **波动率调整**：近 20 日日收益标准差 >3% 的高波动股，单月方向随机性强，动量信号权重应下调。
5. **A股月度特性**：极端单月大涨（如 20 日超额 >15%）后，下月出现均值回归的概率不低，避免线性外推。

## 常见陷阱

- 只看个股涨跌不与指数对比（题目要的是**相对**沪深300 的方向）
- 用未复权价格跨越除权除息日计算收益（必须用 close_qfq）
- 把长窗口强动量直接外推到未来 20 日，忽视短期过热回调
