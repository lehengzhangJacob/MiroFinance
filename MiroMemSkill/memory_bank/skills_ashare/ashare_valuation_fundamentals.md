## 步骤

1. **估值分位**：用 ashare_valuation 取 PE-TTM/PB 及其近 120 日分位：
   - 分位 <20% 且股价近期止跌 → 均值回归上行概率增加；
   - 分位 >80% 且动量转弱 → 回调风险大；
   - 银行/电力等低估值行业看 PB 与股息逻辑，成长行业（电池、汽车）更看 PE 与业绩增速匹配。
2. **财务点时纪律**：ashare_financials 只返回**公告日 <= as_of** 的报告；引用时确认 ann_date，绝不能引用 as_of 之后才披露的季报/年报。
3. **增速与估值匹配**：营收/净利同比（or_yoy、netprofit_yoy）连续下滑 + 高估值 = 戴维斯双杀风险；增速改善 + 低分位 = 修复机会。
4. **财报窗口效应**：as_of 落在财报密集披露期（4月、8月、10月）时，注意「最近一期报告是否即将被新报告取代」，市场可能已提前定价。
5. **权重建议**：20 个交易日的短周期里，估值是慢变量——仅作方向修正（约 30% 权重），动量与资金面为主（约 70%）。

## 常见陷阱

- 引用公告日晚于 as_of 的财务数据（未来函数，直接作废）
- 跨行业直接比较 PE 绝对值（白酒 vs 银行没有可比性）
- 把年报业绩当作次月股价的决定因素（短周期动量往往更强）
