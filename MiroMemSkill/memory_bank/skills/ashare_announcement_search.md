## 步骤

1. **确定实体**：股票代码/公司全称/Wind ticker；中文名用全称搜索。
2. **检索路径**：
   - cninfo.com.cn (巨潮资讯) → 公告/定期报告
   - sse.com.cn / szse.cn 交易所披露
   - 公司 IR 页面投资者关系
3. **筛选报告类型**：年报/半年报/季报/临时公告；按题目要求的时间范围过滤。
4. **提取关键字段**：营收、净利润、EPS、分红、持股比例等；记录报告期与公告日期。
5. **交叉验证**：至少两个来源一致后再作答。

## 常见陷阱

- 用了预告/快报而非正式年报数据
- 混淆报告期与公告发布日期
- 港股/A股代码混用
