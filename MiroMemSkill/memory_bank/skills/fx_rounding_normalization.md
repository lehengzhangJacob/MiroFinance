## 步骤

1. **识别格式要求**：百分比 vs 小数(12.45% vs 0.1245)、千分位、小数位数。
2. **汇率选取**：使用题目指定日期或 as-of 日期的官方中间价/收盘价；注明来源。
3. **换算顺序**：先统一币种再计算；避免重复换算。
4. **舍入时机**：中间步骤保留足够精度，最后一步才按题目要求舍入。
5. **容差判断**：若标准答案含"允许误差/rounding errors allowed"，在容差范围内即视为正确。

## 常见陷阱

- 12.98% 写成 12.98(缺%) 或 0.1298(格式错)
- 汇率用了错误日期
- 整数题给了过多小数位导致判错
